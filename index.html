Public Projects

AITownCrier   Github Twitter

AITownCrier combines a fine-tuned stable diffusion model with OpenAI's chatGPT to generate and curate unique visuals and narratives from scraped news stories worldwide. With the integrated system set up on AWS, these reimagined news stories are automatically posted to Twitter.

Semantic segmentation from sentences   Github

Adopting the approach outlined in The Devil is in the Details paper, I finetuned a Segformer model to deliver zero-shot segmentation. By converting image labels to sentence embeddings, the model could identify various architectural components across diverse building styles without prior exposure to these elements.
